About

Tube Chaser is an Android application for the London Underground.

It was created because there was a lack of good quality, free and open source apps for the Tube in the Android Market.

Current Features

Tube Chaser aims to be as quick and clean as possible for getting to the data you want.

The main features of Tube Chaser are:

  • Quickly see tube line status
  • Browse or search for stations
  • Favorite (star) stations for quick reference
  • Find nearby stations by Android location services
  • Integrated search for stations throughout the app, utilising the 'Search' button on the device
  • See stations on Google Maps
  • Clean user interface based on Google recommended guidelines
  • Open Source available under a GPLv3 license

Upcoming Features

Some features that will be implemented in future versions:

  • Tube map
  • DLR support
  • Live departures/Tube status widget
  • Switch to XML feed to tube line status
  • This weekend information
